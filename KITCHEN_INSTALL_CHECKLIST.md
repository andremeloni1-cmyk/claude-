# Kitchen Install Checklist

A reusable tracker for kitchen installation jobs. Use the **Overview** table to
see what's left at a glance, and the **Room Checklist** sections to track
detailed tasks room by room.

## How to use this

1. Add a row to the Overview table for every room/area in the job.
2. Copy the **Room Checklist Template** block once per room, rename the
   heading, and check items off as they're completed.
3. Anything left unchecked is what's still outstanding for that room.
4. Use the **Notes / What's Left** line at the bottom of each room to call out
   anything custom (special orders, client requests, damage, etc).

## Room Checklist Template

Copy everything between the `---` lines below for each room, then rename
the heading (e.g. `## Main Kitchen`).

---

### [Room Name]

**Status:** Not Started / In Progress / Complete
**Target completion date:**

#### Demo & Prep
- [ ] Remove old cabinets / countertops / appliances
- [ ] Cap or disconnect plumbing & electrical
- [ ] Patch and prep walls / subfloor
- [ ] Confirm on-site measurements match plans

#### Cabinets
- [ ] Base cabinets installed
- [ ] Wall cabinets installed
- [ ] Island / peninsula cabinets installed
- [ ] Fillers, panels, crown molding installed
- [ ] Hardware (knobs/pulls) installed
- [ ] Doors and drawers aligned/adjusted

#### Countertops
- [ ] Templating complete
- [ ] Countertops fabricated
- [ ] Countertops installed
- [ ] Sink cutout / seams finished
- [ ] Caulking and sealing complete

#### Plumbing
- [ ] Sink installed
- [ ] Faucet installed
- [ ] Garbage disposal installed
- [ ] Dishwasher connected
- [ ] Fridge water line connected
- [ ] Leak check complete

#### Electrical
- [ ] Outlets / GFCIs installed
- [ ] Under-cabinet lighting installed
- [ ] Pendant / fixture wiring complete
- [ ] Appliance circuits connected
- [ ] Switches / dimmers tested

#### Appliances
- [ ] Range / cooktop installed
- [ ] Oven installed
- [ ] Range hood / vent installed
- [ ] Refrigerator in place
- [ ] Dishwasher in place
- [ ] Microwave installed

#### Backsplash & Finishes
- [ ] Backsplash tile installed
- [ ] Grout / caulk finished
- [ ] Paint touch-up complete
- [ ] Trim / molding installed

#### Flooring
- [ ] Flooring installed
- [ ] Transitions / thresholds installed
- [ ] Floor protected and cleaned

#### Final Punch List
- [ ] All doors/drawers operate smoothly
- [ ] All appliances tested
- [ ] Final clean
- [ ] Walkthrough / sign-off
- [ ] Photos taken

#### Notes / What's Left

-
