# AM — Andre Meloni Wedding Photography Logo

A minimal, elegant brand mark: the two peaks form an **M**, and the gold
crossbar on the first peak turns it into an **A** — the two initials share
a single stroke. A gold sparkle accent sits above, inside a thin double ring.
Wordmark set in Cormorant Garamond (converted to outlines, so the SVGs render
identically everywhere with no font installed).

## Files

| File | Use |
|------|-----|
| `svg/am-logo-primary.svg` | Main logo for light backgrounds (transparent) |
| `svg/am-logo-dark.svg` | Main logo for dark backgrounds (charcoal panel) |
| `svg/am-monogram.svg` | Monogram badge only — social avatars, favicons, stamps |
| `svg/am-monogram-dark.svg` | Monogram badge on charcoal |
| `svg/am-watermark-white.svg` | White monogram at 85% opacity — overlay on photos |
| `png/…` | High-resolution PNG renders of each (previews/ready to use) |

SVGs are the masters — they scale to any size with no quality loss
(business cards, website, album embossing, signage).

## Brand colors

| Color | Hex | Role |
|-------|-----|------|
| Charcoal | `#2B2725` | Primary mark & text |
| Gold | `#B0905A` | Accent (crossbar, sparkle, tagline) |
| Ivory | `#F7F4EE` | Background / reversed mark |

## Typeface

Wordmark: **Cormorant Garamond** (free, Google Fonts) — use it for headings
on the website and stationery to match the logo.
