# AM — Andre Meloni Wedding Photography Logo

A minimalist mark: two thin peaks form an **M**, and the single gold
crossbar on the first peak turns it into an **A** — both initials drawn
with one shared stroke. No containers or ornament; just the mark and
airy, widely-tracked type. Wordmark set in Cormorant Garamond, converted
to outlines so the SVGs render identically everywhere with no font installed.

## Files

| File | Use |
|------|-----|
| `svg/am-logo-primary.svg` | Main logo for light backgrounds (transparent) |
| `svg/am-logo-dark.svg` | Main logo on charcoal, for dark backgrounds |
| `svg/am-monogram.svg` | Mark only — social avatars, favicons, stamps |
| `svg/am-monogram-dark.svg` | Mark only, on charcoal |
| `svg/am-watermark-white.svg` | White mark + name at 85% opacity — overlay on photos |
| `png/…` | High-resolution PNG renders of each |

SVGs are the masters — they scale to any size with no quality loss
(business cards, website, album embossing, signage).

## Brand colors

| Color | Hex | Role |
|-------|-----|------|
| Charcoal | `#2B2725` | Primary mark & text |
| Gold | `#B0905A` | Single accent (the A crossbar) |
| Ivory | `#F7F4EE` | Background / reversed mark |
| Taupe | `#9A9287` | Tagline |

## Typeface

Wordmark: **Cormorant Garamond** (free, Google Fonts) — use it for headings
on the website and stationery to match the logo.
